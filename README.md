
## 图形学核心笔记 \| Computer Graphics Notes

高质量图形学核心知识点总结笔记，聚焦 **MVP矩阵变换、光栅化、光线追踪、BRDF与渲染方程** 等硬核内容，从原理推导到实践理解一站式梳理，内容比较复杂，请按需查看，全程支持LaTex公式渲染。



## 📖 笔记内容

### 1\. MVP矩阵变换 光栅化与深度缓存（Z\-buffer）详细总结

核心聚焦渲染管线基础变换与光栅化核心流程，帮你吃透图形学基础渲染逻辑，包含大量公式推导，适配GitHub LaTex渲染。

- MVP矩阵完整拆解：模型矩阵（Model）、视图矩阵（View）、投影矩阵（Projection）的数学推导、几何意义与实际作用

- 光栅化渲染管线全流程：从顶点变换到片段着色的每一步细节解析

- Z\-buffer（深度缓存）：原理、核心算法实现、消除隐藏面的逻辑的细节，附关键公式说明

- 图形学基础渲染必备知识点，配套逻辑梳理，避免概念混淆

### 2\. 从零到精通：光线追踪、BRDF与渲染方程 终极学习笔记

从基础到进阶，打通光追渲染底层逻辑，重点解析核心公式与物理意义，适合进阶学习与面试深度学习。

- 光线追踪核心原理：光线生成、相交检测、递归采样的经典算法与实现思路

- BRDF（双向反射分布函数）：定义、核心性质、常用模型（Lambert、Phong等）的公式推导与理解

- 渲染方程：数学推导、物理意义、积分形式解析，以及在实际渲染中的应用

- 进阶拓展：光追与光栅化的对比、实际项目中的应用技巧
